📋 Developer Implementation Notes (Remove from production)

Form & Validation

  • ZIP → auto-lookup city/state via ziptastic.com or similar free API
  • Phone validation: Twilio Lookup API or Telesign on blur (not on keystroke)
  • Email validation: ZeroBounce or IPQualityScore API call
  • Phone format mask: US format (555) 000-0000 auto as user types
  • All fields localStorage cached on change for repeat-visit prefill

Compliance (Critical)

  • TrustedForm script embedded in <head> — must load before form renders
  • Jornaya LeadiD script alongside TrustedForm
  • TCPA text must be visible and unobscured on Step 4 before submit
  • Buyer name list linked from TCPA must match actual ping-post buyers
  • Certificate URL returned from TrustedForm appended to lead payload

Lead Distribution

  • On submit → LeadProsper ping-post API (real-time)
  • Fraud check: Anura before submit is accepted
  • DNC scrub: Blacklist Alliance on phone field
  • Duplicate suppression: check hash of email+phone against last 30 days
  • Lead cert + fields → GoHighLevel CRM webhook

Analytics & Tracking

  • Everflow pixel fired on Step 4 load and on submit success
  • GA4 events: form_start, step_complete (each step), form_submit
  • Track drop-off per step to optimize CVR
  • UTM params preserved in localStorage across steps
  • RPM = lead revenue / sessions × 1000 (primary dashboard KPI)

Performance (Core Web Vitals)

  • Target LCP <2.5s — hero form must paint first
  • Fonts preloaded in <head> with rel="preconnect"
  • Images (carrier logos): use SVG or WebP, lazy-load below fold
  • Cloudflare CDN in front — assets cached at edge
  • Form state in localStorage to survive page refreshes

Outreach Stack (Post-Submit)

  • SMS drip: Textdrip (GHL-triggered, day 0 / day 2 / day 5)
  • Email: Brevo transactional confirmation → SendGrid for drip sequence
  • AI Call: Fluents — trigger call within 60 sec of submit
  • RVM: Drop Cowboy — day 1 if no contact made
  • All outreach must honor DNC and STOP/unsubscribe from Blacklist Alliance
